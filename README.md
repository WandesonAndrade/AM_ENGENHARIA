AM Engenharia | Portfólio Digital
Este repositório contém a infraestrutura e o código-fonte do portal oficial da AM Engenharia. O site foi desenvolvido para servir como uma vitrine de nossas soluções, destacando nossa expertise técnica, gestão de projetos e compromisso com a qualidade na construção civil.

🚀 Sobre o Projeto
O portal integra tecnologias modernas para proporcionar uma experiência fluida aos nossos clientes e parceiros. Combinamos desenvolvimento web de alta performance com a integração de inteligência artificial, permitindo um atendimento automatizado e eficiente para consultas sobre nossos serviços e cronogramas de obra.

🛠️ Tecnologias Utilizadas
Desenvolvimento: Node.js

Inteligência Artificial: Google Gemini API

Versionamento: Git e GitHub
